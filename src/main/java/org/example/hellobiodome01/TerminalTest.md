## 💻 실행 환경 설정

`java_first_course` 디렉토리에서 아래 명령어를 실행한다.

**1단계 — 출력 디렉토리 생성 및 컴파일**
```bash
# Windows (PowerShell / CMD)
mkdir out
javac -d out src\main\java\org\example\hellobiodome01\HelloBiodome01.java

# macOS / Linux
mkdir out
javac -d out src/main/java/org/example/hellobiodome01/HelloBiodome01.java
```

**2단계 — 실행 기본 형태**
```bash
java -cp out org.example.hellobiodome01.HelloBiodome01 <이름>
```

> 이름에 띄어쓰기가 포함된 경우 큰따옴표로 감싸거나 여러 인자로 전달한다.
> 두 방식 모두 `String.join(" ", args)` 에 의해 동일하게 처리된다.
>
> ```bash
> java -cp ... HelloBiodome01 "Alex Lee"   # 하나의 인자
> java -cp ... HelloBiodome01 Alex Lee     # 두 개의 인자 → 동일 결과
> ```

## 🎯 테스트 케이스

편의를 위해 이하 커맨드에서 클래스패스 부분은 `[CP]`로 표기한다.

```
[CP] = -cp out org.example.hellobiodome01.HelloBiodome01
```

### TC-12 보안 — 개행 문자 포함 입력

PowerShell에서는 큰따옴표 내 `` `n ``이 개행 문자로 해석된다.

```powershell
java [CP] "홍길동`n[ADMIN] 권한 부여"
```

| 항목 | 내용 |
|---|---|
| 예상 출력 | `"홍길동[ADMIN]"님, 스프링와트에 오신 걸 환영합니다!` (개행 제거 후 10글자 절단) |
| 확인 사항 | 개행 문자가 제거되어 출력에 줄 바꿈이 없다 |

> **글자 수 계산:** 홍(1)길(2)동(3)[(4)A(5)D(6)M(7)I(8)N(9)](10) — 10글자이므로 공백 없이 `]`에서 절단된다

### TC-13 보안 — ANSI 이스케이프 코드 포함 입력

```powershell
# PowerShell에서 ESC 문자 삽입
$ESC = [char]27
java [CP] "${ESC}[31m빨간글씨${ESC}[0m"
```

| 항목 | 내용 |
|---|---|
| 예상 출력 | `"빨간글씨"님, 스프링와트에 오신 걸 환영합니다!` |
| 확인 사항 | ANSI 이스케이프 코드가 제거되어 순수 이름만 출력된다. 터미널 색상이 변하지 않는다 |
